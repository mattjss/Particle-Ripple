# Particle Ripple Metal Shaders

SwiftUI app with an interactive Metal shader ripple effect. Tap anywhere to trigger the ripple.

## Open the Project

1. In Finder, go to the **Particle Ripple Metal Shaders** folder
2. Double-click **Particle Ripple Metal Shaders.xcodeproj**
3. In Xcode, select a simulator (e.g. iPhone 16)
4. Press âŒ˜R to build and run

## Requirements

- Xcode 15+
- iOS 17+
- SwiftUI

## Files

| File | Purpose |
|------|---------|
| RippleEffect.metal | Metal shader for the ripple |
| RippleModifier.swift | SwiftUI modifiers |
| ContentView.swift | Main view with tap-to-ripple |
| ParticleRippleMetalShadersApp.swift | App entry point |
